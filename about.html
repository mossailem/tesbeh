About Tesbeh

Tesbeh is progressive web application (PWA) that help you do dzikrullah. Whenever you don't bring your tasbih, just open this application.

As Tesbeh is PWA, you can install it in your desktop or smartphone.

Technology Stack

  • HTML
  • CSS
  • Vanilla Javascript
  • Bootstrap Icons v1.11.2
